color:

Sets the text color.

Values: Named colors (like red, blue), hexadecimal color codes, RGB values, etc.

background-color:

Sets the background color of an element.

Values: Named colors (e.g. 'red', 'blue'), hexadecimal color codes, RGB values, etc.

font-family:

defines the font for the text.

Values: 'Font names, generic font families (e.g., 'serif', 'sans-serif').

font-size:

Sets the text size.

Values: Pixels (px), ems (em), percentages (%), etc.

font-weight:

Sets the boldness of the text.

Values: ' normal', 'bold', numeric values (e.g. '700').

font-style:

Sets the style of the text.

Values: 'normal', 'italic', 'oblique'

text-decoration:

Adds decorations to text.

Values: 'none', 'underline', 'overline', 'line-through'.

text-align:

Specifies text alignment.

Values: 'left', 'right','center','justify'.

margins and padding:

margin:Sets margins around an element
padding: Sets padding within an element

Values: 'Length values (px, em)', 'percentages (%)', 'auto'

border:

Sets all border properties

Values: 'border-width', 'border-style', 'border-color'

width and height:

width: Sets the width of an element
height: Sets the height of an element

Values: 'Pixels(px), 'percentages (%)', 'auto'

display:

Specifies how an element is displayed.

Values: 'block', 'inline', 'inline-block', 'none'

position:

Specifies the positioning method.

Values: 'static', 'relative', 'absolute', 'fixed'

float:

Specifies how an element should float.

Values: 'left', 'right', 'none'

background-image:

Sets a background image.

Values: 'URL'

background-repeat:

Defines how background images are repeated.

Values: 'repeat', 'no-repeat', 'repeat-y', 'repeat-x'

background-size:

Specifies the size of background images..

Values: px

visibility:

Controls the visibility of an element.

Values: 'visible', 'hidden'

opacity:

Sets the opacity of an element:

Values: Numeric values between 0 (transparent) and 1 (opaque)

transform:

Applies 2D and 3D tranformations to element.

Values: 'rotate(deg)', 'skewX(deg)', 'skewY(deg)'

transform-origin:

Sets the origin point for transformations.

Values: 'x-axis', 'y-axis'

box-sizing:

Specifies how an element's is box value is calculated.

Values: 'content-box', 'border-box'

overflow:

Defines how content that overflows an element's box is handled.

Values: 'visible', 'hidden', 'scroll', 'auto'

list-style-type:

Sets the list item marker type.

Values: 'circle', 'square', 'upper-roman', 'lower-alpha'

list-style-image:

Sets an image as the list item marker.

Values: 'URL'
